# Backend Environment Variables

Render backend web service'e şu environment variables'ları ekleyin:

## Environment Variables

```
NODE_ENV = production

DATABASE_URL = [Database sayfasından kopyaladığınız Internal Database URL]

JWT_SECRET = 32d65409ee8422ad4128bd25ba378565b4d6cb88253c548ab19c3701c11ee9e0

CLIENT_URL = https://meethub.vercel.app
```

## Adımlar

1. ✅ Database sayfasında "Internal Database URL" kopyalayın
2. ✅ Backend web service sekmesine geri dönün
3. ✅ "Environment Variables" bölümüne yukarıdaki 4 değişkeni ekleyin
4. ✅ "Create Web Service" butonuna tıklayın
5. ⏳ Deployment başlayacak (3-5 dakika)

Deployment tamamlandığında bana **"backend deploy oldu"** yazın!
